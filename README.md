# ImageProcessor
An application which focused on fundamental image processing technique,it was developed on qt platform and halcon library.

一个基于qt开发的包含各种基础图像处理技术的桌面应用，图像处理算法基于halcon，有直接调用halcon脚本和执行halcon C++代码方式。
